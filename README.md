# willycamp.com

Info and registration page for a summer ski camp.

## Developer Setup

You're gonna need [Node](https://nodejs.org/en/). Then:

```bash
# clone the repo and change directories
git clone https://github.com/cadejscroggins/willycamp.com.git
cd willycamp.com

# install packages
npm i

# start the dev server
npm start
```
